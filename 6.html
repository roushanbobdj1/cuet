Topic 1 — Rate of Change of Quantities
HOT ▼

Derivative dy/dx ka matlab hai — y ki rate of change w.r.t. x. Ye chapter ka sabse basic aur seedha topic hai.

Core Concept
dy/dx = Rate of change of y with respect to x

dy/dt = Rate of change of y with respect to time t

dy/dx = (dy/dt) ÷ (dx/dt) ← Chain Rule se

Common Real-Life Rate of Change Problems

Circle / Sphere / Cylinder Formulas
Area of Circle A = πr² → dA/dt = 2πr · dr/dt
Volume of Sphere V = (4/3)πr³ → dV/dt = 4πr² · dr/dt
Surface Area of Sphere S = 4πr² → dS/dt = 8πr · dr/dt
Volume of Cylinder V = πr²h → dV/dt = πr²·dh/dt + 2πrh·dr/dt
⚡ Step-by-Step Method

Step 1: Jo quantity change ho rahi hai uska formula likho (A, V, S etc.)
Step 2: Dono sides ko 't' ke respect mein differentiate karo
Step 3: Given values substitute karo
Step 4: Answer nikalo (units dhyan rakho!)

📌 Important Units
  • Rate of change of Area → cm²/s, m²/s
  • Rate of change of Volume → cm³/s, m³/s
  • Rate of change of radius/length → cm/s, m/s

Topic 2 — Increasing & Decreasing Functions
HOT ▼

Basic Definitions

Ek function f(x) kisi interval mein:

Strictly Increasing
x₁ < x₂ → f(x₁) < f(x₂) [in interval (a,b)]
Condition: f'(x) > 0 for all x ∈ (a,b)

Strictly Decreasing
x₁ < x₂ → f(x₁) > f(x₂) [in interval (a,b)]
Condition: f'(x) < 0 for all x ∈ (a,b)

Steps to Find Intervals

  1. f'(x) nikalo
  2. Critical points find karo: f'(x) = 0 se x ka value
  3. Number line par mark karo, intervals banao
  4. Har interval mein f'(x) ka sign check karo
  5. +ve → Increasing | −ve → Decreasing
f'(x) SignNature of f(x)Graph
f'(x) > 0Increasing ↑Sloping up ↗
f'(x) < 0Decreasing ↓Sloping down ↘
f'(x) = 0Stationary PointFlat (—)
⚡ Quick Sign Method

f'(x) ko factor form mein likho. Phir wavy curve method use karo — roots ke beech signs alternate karte hain (positive → negative → positive...). Highest power ke sign se start karo!

⚠️ Common Mistake

Critical point par function ka value directly extreme nahi hota — pehle sign change check karo! f'(x)=0 only means stationary point.

Topic 3 — Tangent & Normal to Curves
HOT ▼

Key Formulas

Slope of Tangent at point (x₁, y₁)
m_tangent = dy/dx |_(x₁,y₁) = f'(x₁)

Slope of Normal at point (x₁, y₁)
m_normal = −1 / f'(x₁) [∵ tangent ⊥ normal]

Equation of Tangent
y − y₁ = f'(x₁) · (x − x₁)

Equation of Normal
y − y₁ = [−1/f'(x₁)] · (x − x₁)

Special Cases

Special Slopes
Tangent ∥ x-axis → f'(x₁) = 0
Tangent ∥ y-axis → f'(x₁) = ∞ (undefined)
Tangent ⊥ x-axis → Normal ∥ x-axis → f'(x₁) = ∞
Tangent passes through origin → y₁ = f'(x₁) · x₁
⚡ Tangent & Normal Trick

1. f'(x₁) = m (slope of tangent)
2. Normal slope = −1/m
3. Both equations use point-slope form: y−y₁ = m(x−x₁)
4. Tangent aur Normal ek hi point se guzarte hain!

📌 Parametric Curves

Agar x = f(t), y = g(t) diya ho to:

dy/dx = (dy/dt)/(dx/dt) = g'(t)/f'(t)

Topic 4 — Approximations using Derivatives
EASY MARKS ▼

Derivatives se approximately kisi quantity ka value nikalna — ye ek easy 1-step question hota hai CUET mein.

Approximation Formula
Δy ≈ dy = f'(x) · Δx

f(x + Δx) ≈ f(x) + f'(x) · Δx

where: Δx = small change in x (given) Δy = approximate change in y

Solved Example Type

Example: Find √25.1 approximately

Let f(x) = √x, x=25, Δx=0.1
f'(x) = 1/(2√x) → f'(25) = 1/10
f(25.1) ≈ f(25) + f'(25)×0.1 = 5 + 0.01 = 5.01

⚡ Approximation Trick

Nearest "perfect" value choose karo (25 for √25.1, 27 for ∛27.3 etc.). Fir Δx = difference between actual and chosen value. 2 steps mein answer!

Topic 5 — Maxima & Minima (Local & Global)
MOST IMPORTANT ▼

Definitions

Local Maximum: f(c) ≥ f(x) for all x near c
Local Minimum: f(c) ≤ f(x) for all x near c
Critical Point: Where f'(c)=0 or f'(c) is undefined

Method 1: First Derivative Test

Sign Change of f'(x) at critical point x=c
+ve to −ve : Local Maximum at x=c
−ve to +ve : Local Minimum at x=c
No sign change : Neither max nor min (point of inflection)

Method 2: Second Derivative Test

At critical point where f'(c) = 0
f''(c) < 0 → Local MAXIMUM
f''(c) > 0 → Local MINIMUM
f''(c) = 0 → Test FAILS → Use 1st derivative test

💡 Trick: f''(c) < 0 means curve concave down (∩) → Max
f''(c) > 0 means curve concave up (∪) → Min
MLMU Rule
Minus (f'' < 0) → Local Max (∩)
Plus (f'' > 0) → Local Min (∪)
Simple hai — Negative mein MAX, Positive mein MIN!

Absolute (Global) Maximum & Minimum

Closed Interval [a, b] Method
Step 1: Find all critical points where f'(x) = 0 in (a,b)
Step 2: Evaluate f(x) at all critical points
Step 3: Evaluate f(a) and f(b) (endpoints)
Step 4: Largest value = Absolute Max, Smallest = Absolute Min
⚡ Application Problems — Optimization

Step 1: Variable define karo (e.g., x = side of square)
Step 2: Objective function banao (Area/Volume/Cost maximize/minimize karna hai)
Step 3: Constraint se ek variable eliminate karo
Step 4: f'(x)=0 solve karo → critical point
Step 5: f''(x) check karo → Max ya Min confirm karo
Step 6: Value calculate karo

Most Common Application Types

  • Rectangle in circle: x²+y²=r² constraint, maximize xy
  • Box without lid: Volume maximize karo given surface area
  • Cylinder in cone: Cone ke andar cylinder ki max volume
  • Sum/Product optimization: x+y=k diya, xy maximize karo
  • Cost minimization: Transportation ya fencing problems
📌 AM-GM Shortcut

Agar x+y=k (constant) diya hai → xy maximize hoga jab x=y=k/2
Agar xy=k (constant) diya hai → x+y minimize hoga jab x=y=√k
Ye shortcut optimization mein bahut fast answer deta hai!

Topic 6 — Concavity & Point of Inflection
MODERATE ▼
Concavity Test
f''(x) > 0 on (a,b) → Concave UP (∪) in (a,b)
f''(x) < 0 on (a,b) → Concave DOWN (∩) in (a,b)

Point of Inflection
Where concavity changes → f''(x) = 0 AND sign of f''(x) changes
(Note: f''(x)=0 alone is not sufficient!)
⚠️ Inflection Point Mistake

f''(c)=0 ka matlab inflection point NAHI hai necessarily. Sign change zaroori hai! Example: f(x)=x⁴ → f''(0)=0 but no sign change → x=0 is NOT inflection point.

Topic 7 — Rolle's & Mean Value Theorem
MODERATE ▼

Rolle's Theorem

Conditions (ALL 3 must hold)
1. f is continuous on [a, b]
2. f is differentiable on (a, b)
3. f(a) = f(b)

THEN: ∃ c ∈ (a, b) such that f'(c) = 0

Mean Value Theorem (MVT / Lagrange's)

Conditions (2 conditions)
1. f is continuous on [a, b]
2. f is differentiable on (a, b)

THEN: ∃ c ∈ (a, b) such that
f'(c) = [f(b) − f(a)] / (b − a)

💡 Geometric meaning: Instantaneous slope = Average slope
Rolle ⊂ MVT
Rolle's theorem is a special case of MVT
where f(a) = f(b) → f'(c) = 0/something = 0
⚡ MVT Question Type — CUET mein

f'(c) = [f(b)−f(a)]/(b−a) formula mein b, a, f(b), f(a) substitute karo → ek equation milega → c solve karo. Bas itna hi! 2 min ka question hai.

⚡ Must-Know Quick Rules

  • f'(x) > 0 → Increasing; f'(x) < 0 → Decreasing
  • f''(c) < 0 → Local Max (concave down ∩); f''(c) > 0 → Local Min (concave up ∪)
  • Tangent slope = f'(x₁); Normal slope = −1/f'(x₁) [product = −1 for ⊥ lines]
  • Approximation: f(x+Δx) ≈ f(x) + f'(x)·Δx
  • MVT: f'(c) = [f(b)−f(a)]/(b−a) — instantaneous = average rate
  • Rolle's = Special MVT jab f(a)=f(b)
  • Point of Inflection: f''(x)=0 AND sign of f'' changes
  • Optimization: Ek variable mein likho → differentiate → f'=0 → f'' check
  • x+y = const → xy max jab x=y | xy = const → x+y min jab x=y
  • Global max/min on [a,b] → Check ALL critical points + both endpoints

A ladder 5m long leans against a wall. Its foot slides away at 2 cm/s. Find rate of descent of top when foot is 4m away.
CUET 2024
A) 8/3 cm/s
B) 2/3 cm/s
C) 4/3 cm/s
D) 3/2 cm/s
1
x²+y²=25 (Pythagoras, ladder=5m), x=4m, dx/dt=2 cm/s

At x=4: 16+y²=25 → y=3m

Negative sign = descending. Rate of descent = 8/3 cm/s

⏱️ Time Allocation

AOD se 6–8 marks expected hain. Rate of change, tangent, increasing/decreasing = easy 2 min each. Max/min optimization = 4–5 min. Total ~20–25 min agar 7 questions hain.

🎯 Priority Order for Last-Minute Prep

  1. Increasing/Decreasing — f' nikalo, sign check karo → Done!
  2. Tangent slope — f' nikalo, x substitute karo → Done!
  3. Rate of Change — Formula likhke t se differentiate karo
  4. Local Max/Min — f'=0, f'' check karo
  5. Approximation — f(x+Δx) ≈ f(x)+f'(x)Δx → easiest formula!
  6. MVT — f'(c)=[f(b)−f(a)]/(b−a) formula apply karo

❌ Top Mistakes — Avoid Karo!

  • f''=0 se directly inflection point mat samjho — sign change bhi chahiye
  • Local Max find karne ke baad value bhool jao — f(x) ka value bhi nikalo!
  • Global max/min mein endpoints miss mat karo
  • Rate of change mein units galat mat likho
  • Normal slope = −1/(tangent slope) — inverse AND negative!
