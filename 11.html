Unit 2 · Data Structures & Algorithms (DSA)
~25% ▾

📚 Stack (LIFO)

  • Push, Pop, Peek/Top operations
  • isEmpty() check
  • Python implementation: list as stack
  • Applications: Undo, Browser Back, Balanced Brackets
  • Infix → Postfix conversion

🚶 Queue (FIFO)

  • Enqueue (rear), Dequeue (front)
  • Circular Queue concept
  • Priority Queue
  • Deque (Double Ended Queue)
  • Applications: CPU scheduling, BFS

🔗 Linked List

  • Node structure: data + next pointer
  • Singly, Doubly, Circular Linked List
  • Insertion: beginning, middle, end
  • Deletion operations
  • Advantages over arrays

🌳 Trees

  • Binary Tree – root, leaf, height
  • BST (Binary Search Tree) – insert, search, delete
  • Tree Traversals: Inorder (LNR), Preorder (NLR), Postorder (LRN)
  • Level Order (BFS)
  • Height & Depth calculation

🔍 Searching Algorithms

  • Linear Search – O(n)
  • Binary Search – O(log n) [sorted array only]
  • Step-by-step trace questions
  • Number of comparisons puchha jaata hai

🔃 Sorting Algorithms

  • Bubble Sort – adjacent swap
  • Selection Sort – min element select
  • Insertion Sort – sorted portion mein insert
  • Merge Sort – divide & conquer O(n log n)
  • Quick Sort – pivot based O(n log n) avg
✅
Tree Traversal Trick: Inorder of BST hamesha sorted order deta hai. Ye rule CUET mein kai baar aaya hai!

💻

Unit 4 · Computer Networks
~15% ▾

🔌 Network Basics

  • LAN, MAN, WAN, PAN
  • Topology: Bus, Star, Ring, Mesh, Tree, Hybrid
  • Network devices: Hub, Switch, Router, Repeater, Modem, Bridge, Gateway
  • Transmission media: Wired (Coaxial, Twisted Pair, Fiber Optic) & Wireless

📦 OSI Model (7 Layers)

  • 7. Application – HTTP, FTP, SMTP, DNS
  • 6. Presentation – Encryption, Compression
  • 5. Session – Session management
  • 4. Transport – TCP, UDP, Port numbers
  • 3. Network – IP, Routing
  • 2. Data Link – MAC Address, Frames
  • 1. Physical – Bits, Cables

🌍 IP Addressing

  • IPv4 – 32 bit, dotted decimal (x.x.x.x)
  • IPv6 – 128 bit, hexadecimal
  • Classes: A (0-127), B (128-191), C (192-223)
  • Subnet Mask, Default Gateway
  • Public vs Private IP
  • DHCP – dynamic IP assignment

🔐 Protocols & Security

  • HTTP / HTTPS (port 80/443)
  • FTP (21), SMTP (25), POP3 (110)
  • DNS – domain to IP resolution
  • TCP vs UDP difference
  • SSL/TLS – secure communication
  • Firewall, Proxy Server

Cybersecurity Topics (CUET 2024-25 mein aaye): Malware (Virus, Worm, Trojan, Ransomware), Phishing, Denial of Service (DoS), Man-in-the-Middle attack, Cookies aur Privacy.

Unit 7 · PYQ Trends & Sample Questions (2022–2025)
▾
🗓 CUET CS 2022
Q: Python mein following code ka output kya hoga?
x = [1,2,3,4,5]; print(x[1:4:2])
✅ Answer: [2, 4] — Slicing with step 2 from index 1 to 3
🗓 CUET CS 2023
Q: Stack mein Push(10), Push(20), Pop(), Push(30), Pop() operations ke baad stack ka top element kya hoga?
✅ Answer: 10 — Trace karo: [10,20] → pop 20 → [10,30] → pop 30 → [10], top = 10
🗓 CUET CS 2024
Q: SQL query: SELECT COUNT(*) FROM Student WHERE Marks > 75 GROUP BY Class HAVING COUNT(*) >= 3 — Ye query kya return karegi?
✅ Answer: Har class ka count jahan Marks > 75 wale students 3 ya zyada hain
🗓 CUET CS 2025
Q: Binary Tree ka Inorder Traversal kya hoga agar tree hai: Root=5, Left=3, Right=7, Left of 3=1, Right of 3=4?
✅ Answer: 1, 3, 4, 5, 7 — (LNR: Left → Node → Right, hamesha BST mein sorted aata hai)
