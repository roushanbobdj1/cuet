Topic 1 — Vector Basics, Types & Position Vectors
BASICS▼

Vector = quantity with both magnitude and direction. Scalar = only magnitude.

TypeDefinitionExample
Zero Vector (0⃗)Magnitude = 00⃗
Unit Vector (â)|â| = 1î, ĵ, k̂
Equal VectorsSame magnitude & directiona⃗ = b⃗
Negative VectorSame mag, opposite dir−a⃗
Collinear VectorsParallel (same/opp dir)a⃗ ∥ b⃗
Coinitial VectorsSame initial pointOA⃗, OB⃗
Coplanar VectorsIn same planea⃗,b⃗,c⃗ coplanar
Position Vector & Section Formula
Position vector of P(x,y,z): OP⃗ = xî + yĵ + zk̂

Section formula — P divides AB in ratio m:n (internally):
P = (m·B⃗ + n·A⃗)/(m+n)

Midpoint of AB: M⃗ = (A⃗ + B⃗)/2
Magnitude of Vector
|a⃗| = |xî+yĵ+zk̂| = √(x²+y²+z²)

Unit vector: â = a⃗/|a⃗|

Direction cosines of a⃗=(a,b,c):
l = a/|a⃗|, m = b/|a⃗|, n = c/|a⃗|
⚡ Unit Vector Trick

â = a⃗/|a⃗| — Bas vector ko uski magnitude se divide karo!
Agar |a⃗| = 1 already ho to wo hi unit vector hai.
î, ĵ, k̂ ke magnitude = 1 always.

Topic 2 — Vector Addition & Scalar Multiplication
MODERATE▼
Vector Addition
a⃗+b⃗: Triangle law / Parallelogram law

Triangle Law: OA⃗ + AB⃗ = OB⃗
Parallelogram Law: |a⃗+b⃗|² = |a⃗|²+|b⃗|²+2|a⃗||b⃗|cosθ

Component wise: (a₁î+a₂ĵ+a₃k̂)+(b₁î+b₂ĵ+b₃k̂) = (a₁+b₁)î+(a₂+b₂)ĵ+(a₃+b₃)k̂
Properties
Commutative: a⃗+b⃗ = b⃗+a⃗
Associative: (a⃗+b⃗)+c⃗ = a⃗+(b⃗+c⃗)
k(a⃗+b⃗) = ka⃗+kb⃗
|a⃗−b⃗|² = |a⃗|²+|b⃗|²−2|a⃗||b⃗|cosθ
📌 Important Results
  • |a⃗+b⃗|² + |a⃗−b⃗|² = 2(|a⃗|² + |b⃗|²) ← Parallelogram law of addition
  • |a⃗+b⃗| = |a⃗−b⃗| iff a⃗ ⊥ b⃗
  • If a⃗+b⃗+c⃗=0⃗, then a⃗,b⃗,c⃗ form a closed triangle

Topic 3 — Dot Product (Scalar Product)
HOT▼
Definition & Formula
a⃗·b⃗ = |a⃗||b⃗|cosθ (θ = angle between vectors)

Component form: a⃗·b⃗ = a₁b₁ + a₂b₂ + a₃b₃

cos θ = a⃗·b⃗ / (|a⃗|·|b⃗|)
Standard Results
î·î = ĵ·ĵ = k̂·k̂ = 1 (same unit vectors)
î·ĵ = ĵ·k̂ = k̂·î = 0 (perpendicular unit vectors)

a⃗·a⃗ = |a⃗|²
a⃗·b⃗ = 0 → a⃗ ⊥ b⃗ (if neither is zero vector)
a⃗·b⃗ = b⃗·a⃗ (Commutative ✓)
a⃗·(b⃗+c⃗) = a⃗·b⃗ + a⃗·c⃗ (Distributive ✓)
Projection of a⃗ on b⃗
Projection = a⃗·b⃗ / |b⃗| = a⃗·b̂

Projection vector = (a⃗·b⃗/|b⃗|²)·b⃗
⚡ Dot Product Shortcut

Perpendicular condition: a⃗·b⃗ = 0
Parallel condition: a⃗×b⃗ = 0 (cross product = 0)
|a⃗|² = a⃗·a⃗ → length square = self dot product

Topic 4 — Cross Product (Vector Product)
HOT▼
Definition
a⃗×b⃗ = |a⃗||b⃗|sinθ · n̂
(n̂ = unit vector ⊥ to both a⃗ and b⃗, Right-hand rule)

Component Form — Determinant Method
a⃗×b⃗ = |î ĵ k̂|
|a₁ a₂ a₃|
|b₁ b₂ b₃|

= î(a₂b₃−a₃b₂) − ĵ(a₁b₃−a₃b₁) + k̂(a₁b₂−a₂b₁)
Standard Results
î×î = ĵ×ĵ = k̂×k̂ = 0 (same vectors → 0)
î×ĵ = k̂, ĵ×k̂ = î, k̂×î = ĵ (cyclic order ↺)
ĵ×î = −k̂, k̂×ĵ = −î, î×k̂ = −ĵ (reverse = negative)

a⃗×b⃗ = −(b⃗×a⃗) ← NOT commutative!
a⃗×b⃗ = 0⃗ ↔ a⃗ ∥ b⃗ (parallel condition)
Magnitude = Area Applications
|a⃗×b⃗| = |a⃗||b⃗|sinθ

Area of Parallelogram = |a⃗×b⃗|
Area of Triangle = (1/2)|a⃗×b⃗|

sinθ = |a⃗×b⃗| / (|a⃗|·|b⃗|)
î ĵ k̂ Cyclic
î→ĵ→k̂→î (cyclic, positive)
î×ĵ=k̂ | ĵ×k̂=î | k̂×î=ĵ
Reverse order → Negative result
⚡ Cross Product 3×3 Determinant Trick

Row 1: î ĵ k̂
Row 2: a₁ a₂ a₃ (first vector)
Row 3: b₁ b₂ b₃ (second vector)
Expand along Row 1 with + − + signs!

Topic 5 — Scalar Triple Product & Coplanarity
IMPORTANT▼
Scalar Triple Product [a⃗ b⃗ c⃗]
[a⃗ b⃗ c⃗] = a⃗·(b⃗×c⃗) = det|a₁ a₂ a₃; b₁ b₂ b₃; c₁ c₂ c₃|

Volume of Parallelepiped
V = |[a⃗ b⃗ c⃗]| = |a⃗·(b⃗×c⃗)|

Volume of Tetrahedron = (1/6)|[a⃗ b⃗ c⃗]|

Coplanar Condition
a⃗, b⃗, c⃗ coplanar iff [a⃗ b⃗ c⃗] = 0
📌 Properties
  • [a⃗ b⃗ c⃗] = [b⃗ c⃗ a⃗] = [c⃗ a⃗ b⃗] (cyclic permutation = same)
  • [a⃗ b⃗ c⃗] = −[b⃗ a⃗ c⃗] (interchange → sign change)
  • [a⃗ a⃗ b⃗] = 0 (two same vectors → 0)

Topic 6 — Linear Programming Problem (LPP) Basics
MOST IMP▼

Key Terminology

TermMeaning
Objective FunctionZ = ax+by (to Maximize or Minimize)
Decision Variablesx, y (values to find)
ConstraintsInequalities: ax+by ≤ c, etc.
Non-negativityx ≥ 0, y ≥ 0 (always!)
Feasible RegionSet of all points satisfying all constraints
Feasible SolutionAny point in feasible region
Optimal SolutionMax or Min value of Z in feasible region
Corner PointsVertices of feasible region polygon

Step-by-Step Method to Solve LPP

1
Objective Function likho: Z = ax+by (Maximize/Minimize)
2
Constraints likho: Saari inequalities + x≥0, y≥0
3
Graph banao: Har constraint ko equality mein badlo (ax+by=c), line draw karo
4
Feasible region shade karo: Origin test karo (substitute x=0,y=0), agar satisfy kare to origin side shade karo
5
Corner points find karo: Lines ke intersection points solve karke nikalo
6
Z calculate karo: Har corner point par Z = ax+by calculate karo
7
Optimal Value: Maximum/minimum Z value = answer
⚡ Corner Point Theorem (Most Important!)

If a feasible region is bounded → Optimal solution ALWAYS occurs at one of the CORNER POINTS

Evaluate Z at ALL corner points → Largest = Max, Smallest = Min

Agar koi corner point nahi → Unbounded feasible region → Check separately!

Chapter 12 — Linear Programming

LPP Structure

ObjectiveZ = ax+by (Max/Min)
Constraintsax+by ≤/≥/= c
Non-neg.x≥0, y≥0 (always!)
OptimalAt corner point of feasible region

Graph Method

x-interceptPut y=0 → x=c/a
y-interceptPut x=0 → y=c/b
Origin testSub(0,0) → check satisfy
Corner ptsSolve intersection equations

⚡ Quick Rules

  • Dot product = 0 → Perpendicular vectors
  • Cross product = 0 → Parallel vectors
  • a⃗×b⃗ = −(b⃗×a⃗) — NOT commutative!
  • |a⃗×b⃗| = area of parallelogram; half = area of triangle
  • Triple product = 0 → Coplanar vectors
  • î×ĵ=k̂, ĵ×k̂=î, k̂×î=ĵ — Cyclic order (positive)
  • LPP optimal always at corner point — Corner Point Theorem
  • Feasible region: Origin test → 0≤c → origin side shade
  • Bounded region → Both max AND min exist
  • Unbounded region → May not have both max and min
  • Corner points = Intersection of constraint lines — solve simultaneously
  • LPP me Z evaluate karo ALL corner points par — ek bhi miss mat karo!
