Topic 1 — Continuity at a Point
MOST IMP▼

Ek function f(x) point x=a par continuous hai agar teen conditions satisfy hon:

Condition 1
f(a) exist kare
(defined ho)
Condition 2
lim(x→a) f(x) exist kare
(LHL = RHL)
Condition 3
lim(x→a) f(x) = f(a)
(limit = value)
Continuity Condition (All 3 must hold)
f continuous at x=a iff:
lim(x→a⁻) f(x) = lim(x→a⁺) f(x) = f(a)

LHL = RHL = f(a)

Left Hand Limit (LHL) & Right Hand Limit (RHL)

How to Calculate
LHL = lim(x→a⁻) f(x) = lim(h→0) f(a−h) [h→0⁺]
RHL = lim(x→a⁺) f(x) = lim(h→0) f(a+h) [h→0⁺]

Substitute x = a−h (for LHL) and x = a+h (for RHL), h→0
⚡ Continuity Check Method — 3 Steps

Step 1: f(a) nikalo (direct substitute karo)
Step 2: LHL nikalo: f(a−h) as h→0
Step 3: RHL nikalo: f(a+h) as h→0
Agar teeno equal → Continuous ✓
Koi bhi different → Discontinuous ✗

Types of Discontinuity

TypeConditionExample
RemovableLHL=RHL ≠ f(a)Hole in graph
JumpLHL ≠ RHLStep function
InfiniteLimit = ∞f(x)=1/x at x=0
OscillatoryLimit doesn't existsin(1/x) at x=0
📌 Continuity of Standard Functions
  • Polynomials — Har jagah continuous
  • Rational f(x)/g(x) — Continuous jahan g(x) ≠ 0
  • sin x, cos x, eˣ, ln x — Apne domain mein continuous
  • |x| — Har jagah continuous (x=0 par bhi!)
  • Greatest Integer [x] — Integers par discontinuous

Algebra of Continuous Functions

If f and g are continuous at x=a, then:
f±g → continuous at a
f·g → continuous at a
f/g → continuous at a (if g(a) ≠ 0)
|f| → continuous at a
f∘g (composite) → continuous at a

Topic 2 — Differentiability
HOT▼
Definition of Derivative (First Principle)
f'(a) = lim(h→0) [f(a+h) − f(a)] / h

LHD = lim(h→0⁺) [f(a−h) − f(a)] / (−h)
RHD = lim(h→0⁺) [f(a+h) − f(a)] / h

f differentiable at a iff LHD = RHD
C → D Rule
Differentiable → Continuous (always true)
Continuous → Differentiable (NOT always true!)
Example: |x| at x=0 — Continuous but NOT Differentiable
⚠️ Most Common CUET Trap!

|x| at x=0: Continuous ✓ but NOT Differentiable ✗
LHD = −1, RHD = +1 → LHD ≠ RHD → Not differentiable
Ye CUET mein har saal kisi na kisi form mein aata hai!

📌 Key Facts
  • Every differentiable function is continuous
  • Converse is NOT true: Continuous ↛ Differentiable
  • Polynomials are differentiable everywhere
  • |x−a| is not differentiable at x=a
  • Corner, cusp, vertical tangent → Not differentiable

Topic 3 — Chain Rule & Differentiation Rules
HOT▼
Chain Rule
d/dx[f(g(x))] = f'(g(x)) · g'(x)

dy/dx = dy/du · du/dx (if y=f(u), u=g(x))
Product Rule
d/dx[u·v] = u·(dv/dx) + v·(du/dx)

Quotient Rule
d/dx[u/v] = [v·(du/dx) − u·(dv/dx)] / v²
⚡ Product Rule Memory — UV Rule

(uv)' = u'v + uv'
"First derivative of Second + Second derivative of First" (FDS + SDF)

Quotient: (u/v)' = (u'v − uv') / v²
"Lo Di Hi minus Hi Di Lo, square the Lo below!"

Important Composite Derivatives

d/dx[sin(ax+b)] = a·cos(ax+b)
d/dx[eᵃˣ⁺ᵇ] = a·eᵃˣ⁺ᵇ
d/dx[ln(f(x))] = f'(x)/f(x)
d/dx[(f(x))ⁿ] = n·(f(x))ⁿ⁻¹·f'(x)
d/dx[aᶠ⁽ˣ⁾] = aᶠ⁽ˣ⁾·ln(a)·f'(x)

Topic 4 — Implicit Differentiation
HOT▼

Jab y ko x se explicit form mein express nahi kar sakte, tab implicit differentiation use karte hain.

Method — Differentiate both sides w.r.t. x
If F(x,y) = 0, then:

d/dx[y] = dy/dx
d/dx[y²] = 2y·(dy/dx) ← Chain rule!
d/dx[y³] = 3y²·(dy/dx)
d/dx[xy] = x·(dy/dx) + y ← Product rule!

Step: Differentiate, collect dy/dx terms, solve

Standard Example

Example: x²+y²=r², find dy/dx

Differentiate: 2x + 2y·dy/dx = 0
→ dy/dx = −x/y

⚡ Implicit Diff Steps

1. Dono sides ko x ke w.r.t. differentiate karo
2. Jab bhi y aaye: dy/dx multiply karo (chain rule)
3. Jab bhi xy aaye: product rule use karo
4. dy/dx ke terms ek side collect karo
5. dy/dx = (x terms)/(y terms) solve karo

Topic 5 — Logarithmic Differentiation
HOT▼

Jab function mein power pe function ho (like xˣ, xˢⁱⁿˣ etc.) tab log differentiation use karo.

Method
Step 1: y = f(x) → Take log both sides: ln y = ln f(x)
Step 2: Differentiate: (1/y)·dy/dx = d/dx[ln f(x)]
Step 3: dy/dx = y · d/dx[ln f(x)]

Key Log Properties Used
ln(uv) = ln u + ln v
ln(u/v) = ln u − ln v
ln(uⁿ) = n·ln u ← Most used!

Standard Results

y = xˣ
ln y = x·ln x
(1/y)dy/dx = ln x + 1
dy/dx = xˣ(1 + ln x)

y = xˢⁱⁿ ˣ
ln y = sin x · ln x
dy/dx = xˢⁱⁿˣ[cos x · ln x + sin x/x]

y = (sin x)ˣ
ln y = x · ln(sin x)
dy/dx = (sin x)ˣ[ln(sin x) + x·cot x]
⚡ Log Diff Trigger

Jab bhi dekhao: (function)^(function) → Logarithmic differentiation karo!
xˣ, x^(sin x), (sin x)ˣ, (tan x)^(cos x) → Sab log diff se karo.

Topic 6 — Parametric Differentiation
IMPORTANT▼
If x=f(t), y=g(t) → Parametric form
dy/dx = (dy/dt) / (dx/dt) = g'(t) / f'(t)

Second derivative d²y/dx²
d²y/dx² = d/dx[dy/dx] = [d/dt(dy/dx)] / (dx/dt)

Standard Parametric Curves

⚡ Parametric Trick

dy/dx = (dy/dt)÷(dx/dt)
Bas y ko t se differentiate karo, x ko t se differentiate karo, divide karo!
"Y prime over X prime" — simple ratio.

Topic 7 — Second & Higher Order Derivatives
IMPORTANT▼
Notation
First derivative: f'(x) = dy/dx = y'
Second derivative: f''(x) = d²y/dx² = y''
Third derivative: f'''(x) = d³y/dx³ = y'''

Method
d²y/dx² = d/dx[dy/dx] → Differentiate dy/dx once more!

Standard Second Derivatives

y=sin x → y'=cos x → y''=−sin x
y=cos x → y'=−sin x → y''=−cos x
y=eˣ → y'=eˣ → y''=eˣ (same always!)
y=xⁿ → y'=nxⁿ⁻¹ → y''=n(n−1)xⁿ⁻²
y=ln x → y'=1/x → y''=−1/x²
📌 Leibniz Theorem (nth derivative)

Dⁿ(uv) = Σ ⁿCᵣ · Dʳu · Dⁿ⁻ʳv (r from 0 to n)
This is useful for proving results like:
If y=sin(ax+b) → yₙ = aⁿ sin(ax+b + nπ/2)

Topic 8 — Inverse Trigonometric Derivatives & Substitutions
HOT▼
Standard Inverse Trig Derivatives
d/dx[sin⁻¹x] = 1/√(1−x²)
d/dx[cos⁻¹x] = −1/√(1−x²)
d/dx[tan⁻¹x] = 1/(1+x²)
d/dx[cot⁻¹x] = −1/(1+x²)
d/dx[sec⁻¹x] = 1/(x√(x²−1))
d/dx[cosec⁻¹x] = −1/(x√(x²−1))

Key Substitutions for Simplification

Simplify before differentiating!
sin⁻¹(2x√(1−x²)): put x=sinθ → 2sin⁻¹x
tan⁻¹(2x/(1−x²)): put x=tanθ → 2tan⁻¹x
tan⁻¹((1−x)/(1+x)): = π/4 − tan⁻¹x
sin⁻¹(2sinx cosx): = 2x
cos⁻¹(2cos²x−1): = 2x (for cos⁻¹(cos2x) = 2x)

After simplification, differentiate the simpler form!
⚡ Inverse Trig Simplification Trick

2sinθcosθ = sin2θ → sin⁻¹(sin 2θ) = 2θ
(1−cos2θ)/2 = sin²θ, 2cos²θ−1 = cos2θ
2tanθ/(1−tan²θ) = tan2θ → tan⁻¹(tan2θ) = 2θ

CUET mein: Pehle substitute karo, simplify karo, phir differentiate karo!
Direct differentiation se complex answer aayega!

Double Angle Identities for Substitution

sin⁻¹(2sinθcosθ) = sin⁻¹(sin2θ) = 2θ (x=sinθ → θ=sin⁻¹x)
cos⁻¹(1−2sin²θ) = cos⁻¹(cos2θ) = 2θ
tan⁻¹(2tanθ/(1−tan²θ)) = tan⁻¹(tan2θ) = 2θ

⚡ Quick Rules

  • Continuous at a: LHL = RHL = f(a) — sabse important condition
  • Differentiable → Continuous, but Continuous ↛ Differentiable
  • |x| at 0: Continuous ✓, NOT differentiable ✗ (LHD=−1 ≠ RHD=+1)
  • Co-functions (cos⁻¹, cot⁻¹, cosec⁻¹) ke derivatives mein always negative
  • Chain rule: d/dx[f(g(x))] = f'(g(x)) · g'(x)
  • Product rule: (uv)' = u'v + uv' → "FDS + SDF"
  • Quotient rule: (u/v)' = (u'v − uv')/v²
  • Log diff trigger: (function)^(function) form dikhte hi
  • Parametric: dy/dx = (dy/dt) ÷ (dx/dt)
  • Implicit: Differentiate both sides, dy/dx terms collect karo
  • Inverse trig simplify: Pehle substitute karo, phir differentiate!
  • d²y/dx² = d/dx of (dy/dx) — ek baar aur differentiate

Continuity PYQs
1
Find k if f(x) = kx+1, x≤5 and f(x) = 3x−5, x>5 is continuous at x=5.
CUET 2022
PYQ · Continuity
A) 1
B) 9/5
C) 3
D) 5/9
1
For continuity at x=5: LHL = RHL = f(5)
2
f(5) = k(5)+1 = 5k+1
3
RHL = lim(x→5⁺)(3x−5) = 3(5)−5 = 10
4
LHL = 5k+1; For continuity: 5k+1 = 10 → k = 9/5
✅ k = 9/5
2
Is f(x) = |x−3| differentiable at x=3? Justify.
CUET 2022
A) No, LHD ≠ RHD
B) Yes, differentiable
C) Not continuous
D) Cannot determine
1
f(x) = |x−3| = {x−3 if x≥3; -(x-3) if x<3}
2
LHD = lim(h→0)[f(3−h)−f(3)]/(−h) = lim[−(−h)−0]/(−h) = −1
3
RHD = lim(h→0)[f(3+h)−f(3)]/h = lim[h−0]/h = +1
4
LHD ≠ RHD → NOT differentiable at x=3
✅ Not differentiable. LHD=−1 ≠ RHD=+1 (Classic |x−a| question!)

4
Find dy/dx if xˢⁱⁿˣ + (sin x)ˣ = 1.
CUET 2023
1

2

→ du/dx = xˢⁱⁿˣ[cosx·lnx + sinx/x]

❌ Top Mistakes — Avoid Karo

  • Continuous ≠ Differentiable — |x| example yaad rakho!
  • Co-functions ka derivative: Always negative (cos⁻¹, cot⁻¹, cosec⁻¹)
  • Inverse trig: Pehle simplify karo, phir differentiate — direct diff mat karo
  • Log diff: +C bhoolna nahi in indefinite; y = eˣ^f(x) mein ln lena
  • Implicit: d/dx(y²) = 2y·dy/dx — chain rule lagana bhoolna
  • Parametric: dy/dx ≠ dy·dx → dy/dt DIVIDED BY dx/dt
🔑 7 Must-Know Formulas for 2026

1. Continuity: LHL=RHL=f(a)
2. Co-fn derivatives: always NEGATIVE (cos⁻¹, cot⁻¹, cosec⁻¹)
3. Chain rule: d/dx[f(g(x))] = f'(g(x))·g'(x)
4. xˣ → xˣ(1+ln x)
5. sin⁻¹(2sinθcosθ) = 2θ → d/dx = 2/√(1−x²)
6. tan⁻¹((1−x)/(1+x)) = π/4−tan⁻¹x → d/dx = −1/(1+x²)
7. Parametric: dy/dx = (dy/dt)/(dx/dt)
