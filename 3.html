Basic Concepts & Definitions

Experiment (प्रयोग): कोई भी process जिसका result निश्चित न हो, जैसे: coin उछालना, dice फेंकना।
Sample Space (S): एक experiment के सभी possible outcomes का set। जैसे coin के लिए S = {H, T}
Event (E): Sample space का कोई subset। जैसे dice में even number = {2,4,6}
Probability: किसी event के होने की संभावना (0 से 1 के बीच)।
Classical Definition of Probability
P(E) = Number of favourable outcomes / Total number of outcomes = n(E) / n(S)
ExperimentSample Spacen(S)
1 Coin उछालना{H, T}2
2 Coins उछालना{HH, HT, TH, TT}4
3 Coins उछालना{HHH, HHT, …}8
1 Dice फेंकना{1,2,3,4,5,6}6
2 Dice फेंकनासभी ordered pairs36
52 Cards में से 1 card52 cards52
हमेशा 0 ≤ P(E) ≤ 1 होता है। P(S) = 1, P(∅) = 0

Types of Events (Events के प्रकार)

Sure Event (निश्चित घटना): P = 1 (जो हमेशा हो) जैसे dice में 1-6 का number आना।
Impossible Event (असंभव घटना): P = 0 जैसे dice में 7 आना।
Complementary Event (A'): A का opposite। A' = S – A
Mutually Exclusive Events: दो events जो एक साथ नहीं हो सकते। A ∩ B = ∅
Exhaustive Events: जिनका union पूरा Sample Space हो। A ∪ B = S
Equally Likely Events: जिनकी probability बराबर हो।
Independent Events: एक का result दूसरे को effect नहीं करता।
Complement Rule
P(A') = 1 – P(A)  |  P(A) + P(A') = 1

Addition Theorem (योग प्रमेय)

General Addition Theorem
P(A ∪ B) = P(A) + P(B) – P(A ∩ B)
Mutually Exclusive Events के लिए
P(A ∪ B) = P(A) + P(B) [क्योंकि P(A ∩ B) = 0]
तीन Events के लिए
P(A ∪ B ∪ C) = P(A)+P(B)+P(C) – P(A∩B) – P(B∩C) – P(A∩C) + P(A∩B∩C)
Venn Diagram बनाकर solve करें — CUET में diagram based प्रश्न आते हैं।
Q1 Addition Theorem Easy
यदि P(A) = 0.4, P(B) = 0.3, P(A ∩ B) = 0.1, तो P(A ∪ B) = ?
(a) 0.5
(b) 0.6
(c) 0.7
(d) 0.8
Solution देखें
P(A ∪ B) = P(A) + P(B) – P(A ∩ B)
= 0.4 + 0.3 – 0.1
= 0.6
✓ Answer: (b) 0.6
Q2 Cards Problem Easy
52 cards में से 1 card randomly निकाली जाती है। King या Heart card आने की probability क्या है?
(a) 4/13
(b) 16/52
(c) 15/52
(d) 17/52
Solution देखें
Kings = 4, Hearts = 13, King of Hearts = 1 (both में count)
P(King ∪ Heart) = 4/52 + 13/52 – 1/52 = 16/52 = 4/13
✓ Answer: (a) 4/13

Conditional Probability (सशर्त प्रायिकता)

Conditional Probability: जब हमें पता हो कि event B हो चुका है, तब A के होने की probability।
Conditional Probability Formula
P(A|B) = P(A ∩ B) / P(B) , जहाँ P(B) ≠ 0

P(B|A) = P(A ∩ B) / P(A) , जहाँ P(A) ≠ 0
P(A|B) ≠ P(B|A) — इन दोनों को confuse मत करें!
Q3 Conditional Probability Medium
एक dice फेंका गया। यदि even number आया है, तो number 2 से अधिक होने की probability क्या है?
(a) 1/3
(b) 2/3
(c) 1/2
(d) 1/6
Solution देखें
B = Even number = {2, 4, 6}, P(B) = 3/6 = 1/2
A = 2 से अधिक = {3,4,5,6}
A ∩ B = even AND > 2 = {4, 6}
P(A ∩ B) = 2/6 = 1/3
P(A|B) = P(A∩B)/P(B) = (1/3)/(1/2) = 2/3
✓ Answer: (b) 2/3
Q4 Two Dice Medium
दो dice फेंके गए। यदि पहला dice 4 दिखाता है, तो दोनों का योग 8 होने की probability क्या है?
(a) 1/6
(b) 1/36
(c) 5/36
(d) 1/4
Solution देखें
B = पहला dice = 4 → (4,1),(4,2),(4,3),(4,4),(4,5),(4,6) = 6 outcomes
A ∩ B = sum=8 AND first=4 → (4,4) = 1 outcome
P(A|B) = 1/6
✓ Answer: (a) 1/6

Multiplication Theorem (गुणन प्रमेय)

Multiplication Theorem
P(A ∩ B) = P(A) × P(B|A)
P(A ∩ B) = P(B) × P(A|B)
Independent Events के लिए
P(A ∩ B) = P(A) × P(B)
Q5 Without Replacement Medium
थैले में 5 लाल और 3 हरी balls हैं। बिना replacement के 2 balls निकाली जाती हैं। दोनों लाल होने की probability?
(a) 5/14
(b) 5/16
(c) 25/64
(d) 10/56
Solution देखें
P(1st Red) = 5/8
P(2nd Red | 1st Red) = 4/7 (क्योंकि 1 लाल ball निकल गई)
P(दोनों लाल) = 5/8 × 4/7 = 20/56 = 5/14
✓ Answer: (a) 5/14

Independent Events (स्वतंत्र घटनाएँ)

Independent Events: A और B independent हैं यदि एक का result दूसरे को affect न करे।
Independence की शर्त
P(A ∩ B) = P(A) · P(B)

P(A|B) = P(A) && P(B|A) = P(B)
Mutually Exclusive ≠ Independent! अगर A और B mutually exclusive हैं और P(A),P(B)>0, तो वे independent नहीं हो सकते।
Q6 Independence Check Medium
P(A) = 1/2, P(B) = 1/3, P(A ∩ B) = 1/6. क्या A और B independent हैं?
(a) हाँ, independent
(b) नहीं, dependent
(c) Mutually Exclusive
(d) Data insufficient
Solution देखें
P(A) × P(B) = 1/2 × 1/3 = 1/6
P(A ∩ B) = 1/6
चूँकि P(A ∩ B) = P(A) × P(B), इसलिए A और B Independent हैं।
✓ Answer: (a) हाँ, independent
Q7 Three Independent Events Hard
A, B, C independent events हैं। P(A)=0.3, P(B)=0.4, P(C)=0.5. कम से कम एक event होने की probability?
(a) 0.79
(b) 0.21
(c) 0.60
(d) 0.83
Solution देखें
P(कोई नहीं) = P(A') × P(B') × P(C')
= (1-0.3)(1-0.4)(1-0.5) = 0.7 × 0.6 × 0.5 = 0.21
P(कम से कम एक) = 1 – 0.21 = 0.79
✓ Answer: (a) 0.79

Bayes' Theorem (बेज प्रमेय)

Bayes' Theorem: यदि B₁, B₂, …, Bₙ sample space के partition हों और A कोई event हो, तो:
Bayes' Theorem Formula
P(Bᵢ|A) = [P(Bᵢ) × P(A|Bᵢ)] / [Σ P(Bⱼ) × P(A|Bⱼ)]

जहाँ j = 1 से n तक
Total Probability Theorem: P(A) = Σ P(Bᵢ) × P(A|Bᵢ)
Q8 Bayes' Theorem Classic Hard
दो थैले हैं। Bag I में 3 लाल, 4 सफेद balls और Bag II में 5 लाल, 2 सफेद balls हैं। एक थैला randomly चुना और एक ball निकाली — वह लाल निकली। Bag I से होने की probability?
Solution देखें
B₁ = Bag I चुना, B₂ = Bag II चुना
P(B₁) = P(B₂) = 1/2
P(Red|B₁) = 3/7, P(Red|B₂) = 5/7

P(B₁|Red) = [P(B₁)·P(Red|B₁)] / [P(B₁)·P(Red|B₁) + P(B₂)·P(Red|B₂)]
= [(1/2)(3/7)] / [(1/2)(3/7) + (1/2)(5/7)]
= (3/14) / (3/14 + 5/14)
= 3/8
✓ Answer: 3/8
Q9 Disease Testing (Bayes') Hard
एक disease 1% लोगों को होती है। Test की accuracy: बीमार है तो positive 99%, स्वस्थ है तो positive 5%. अगर test positive आया, तो actually बीमार होने की probability?
Solution देखें
D = बीमार, P(D) = 0.01, P(D') = 0.99
P(+|D) = 0.99, P(+|D') = 0.05

P(D|+) = [P(D)·P(+|D)] / [P(D)·P(+|D) + P(D')·P(+|D')]
= [0.01 × 0.99] / [0.01×0.99 + 0.99×0.05]
= 0.0099 / (0.0099 + 0.0495)
= 0.0099 / 0.0594
≈ 0.1667 ≈ 16.67%
✓ Answer: ≈ 1/6

Random Variable & Probability Distribution

Random Variable (X): एक variable जो sample space के हर outcome को एक numerical value देता है।
Discrete Random Variable: जो गिनने योग्य values ले सके।
Probability Distribution Table: हर value की probability दिखाता है।
महत्वपूर्ण शर्तें
Σ P(xᵢ) = 1 (सभी probabilities का sum = 1)
0 ≤ P(xᵢ) ≤ 1 (हर probability 0 और 1 के बीच)
Mean (Expected Value)
E(X) = μ = Σ xᵢ · P(xᵢ)
Variance & Standard Deviation
Var(X) = σ² = Σ xᵢ² · P(xᵢ) – [E(X)]²
σ = √Var(X)
Q10 Mean of Distribution Medium
निम्न probability distribution दी है:
X: 0, 1, 2, 3
P(X): 0.1, 0.3, 0.4, 0.2
E(X) = ?
(a) 1.7
(b) 1.5
(c) 2.0
(d) 1.9
Solution देखें
E(X) = 0×0.1 + 1×0.3 + 2×0.4 + 3×0.2
= 0 + 0.3 + 0.8 + 0.6
= 1.7
✓ Answer: (a) 1.7

Binomial Distribution (द्विपद बंटन)

Binomial Distribution: जब n independent trials हों, हर trial में success की probability p (और failure q = 1-p) हो।
Binomial Probability Formula
P(X = r) = ⁿCᵣ × pʳ × qⁿ⁻ʳ

जहाँ: n = trials, r = successes, p = P(success), q = 1–p
Mean, Variance, SD of Binomial Distribution
Mean = μ = np
Variance = σ² = npq
Standard Deviation = σ = √(npq)
Binomial distribution के लिए: n fixed होना चाहिए, हर trial independent होनी चाहिए, p constant होनी चाहिए।
Q11 Binomial Probability Medium
एक fair coin 6 बार उछाली जाती है। ठीक 4 बार Head आने की probability?
(a) 15/64
(b) 6/64
(c) 1/16
(d) 3/8
Solution देखें
n=6, r=4, p=1/2, q=1/2
P(X=4) = ⁶C₄ × (1/2)⁴ × (1/2)²
= 15 × (1/16) × (1/4)
= 15/64
✓ Answer: (a) 15/64
Q12 Binomial Mean Easy
Binomial distribution में n=20, p=0.4. Mean और Variance क्या होंगे?
(a) 8, 4.8
(b) 8, 3.2
(c) 12, 4.8
(d) 4, 2.4
Solution देखें
Mean = np = 20 × 0.4 = 8
q = 1 – 0.4 = 0.6
Variance = npq = 20 × 0.4 × 0.6 = 4.8
✓ Answer: (a) 8, 4.8
Q13 At Least One Success Medium
एक shooter की accuracy 70% है। वह 5 बार shoot करता है। कम से कम एक बार hit करने की probability?
Solution देखें
p = 0.7, q = 0.3, n = 5
P(कम से कम 1 hit) = 1 – P(0 hit)
P(X=0) = ⁵C₀ × (0.7)⁰ × (0.3)⁵ = (0.3)⁵ = 0.00243
P(कम से कम 1) = 1 – 0.00243 = 0.99757
✓ Answer: 0.99757 ≈ 99.76%

Q20 Committee Problem Hard
5 पुरुष और 3 महिलाओं में से 4 का committee बनाना है। कम से कम 1 महिला होने की probability?

Q21 Birthday Problem Variation Hard
यदि P(A) = 0.6, P(B) = 0.5, P(A ∪ B) = 0.8 तो P(A|B) क्या है?
(a) 0.3
(b) 0.6
(c) 0.5
(d) 0.4
Solution देखें
P(A ∩ B) = P(A) + P(B) – P(A∪B) = 0.6+0.5–0.8 = 0.3
P(A|B) = P(A∩B)/P(B) = 0.3/0.5 = 0.6
✓ Answer: (b) 0.6
Q22 Binomial Hard Hard
Binomial distribution में Mean = 4 और Variance = 3 है। p और n क्या हैं?
Solution देखें
np = 4 ... (i)
npq = 3 ... (ii)
(ii)/(i): q = 3/4, इसलिए p = 1/4
n = 4/p = 4/(1/4) = 16
अतः n = 16, p = 1/4
✓ Answer: n = 16, p = 1/4
Q23 Mixed Probability Hard
तीन students A, B, C की exam pass करने की probability क्रमशः 1/2, 1/3, 1/4 है। केवल A pass करे (B और C fail) की probability?
(a) 1/4
(b) 1/6
(c) 1/8
(d) 3/24
Solution देखें
P(A pass, B fail, C fail)
= P(A) × P(B') × P(C')
= (1/2) × (2/3) × (3/4)
= 6/24 = 1/4
✓ Answer: (a) 1/4
