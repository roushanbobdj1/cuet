Topic 1 — Integration Basics & Standard Integrals
MOST IMP ▼

Integration, differentiation ka reverse process hai. Agar d/dx[F(x)] = f(x), to ∫f(x)dx = F(x) + C.

Fundamental Theorem
∫f(x)dx = F(x) + C where d/dx[F(x)] = f(x)
C = Constant of Integration (kabhi mat bhulo!)

📋 Standard Integrals Table — RATTO LO!

∫ f(x) dx= Result + CTrick / Note
∫ xⁿ dxxⁿ⁺¹/(n+1) + Cn ≠ −1
∫ 1/x dxln|x| + Cn = −1 case
∫ eˣ dxeˣ + Ceˣ ka integral eˣ hi!
∫ aˣ dxaˣ/ln(a) + Ca > 0, a ≠ 1
∫ sin x dx−cos x + CNegative aata hai!
∫ cos x dxsin x + CPositive
∫ sec²x dxtan x + Csec²x = derivative of tan
∫ cosec²x dx−cot x + CNegative!
∫ sec x tan x dxsec x + Cderivative of sec x
∫ cosec x cot x dx−cosec x + CNegative!
∫ 1/√(1−x²) dxsin⁻¹x + COr −cos⁻¹x + C
∫ −1/√(1−x²) dxcos⁻¹x + C
∫ 1/(1+x²) dxtan⁻¹x + COr −cot⁻¹x + C
∫ 1/(x√(x²−1)) dxsec⁻¹x + CRare but possible
∫ √x dx(2/3)x^(3/2) + Cx^(1/2) → power rule
∫ 1/√x dx2√x + Cx^(-1/2) case
⚡ Sin/Cos Trick — Negative Signs

∫sin = −cos (negative) | ∫cos = +sin (positive)
∫cosec² = −cot (negative) | ∫sec² = +tan (positive)
Yaad karo: Sin aur cosec² ke integrals mein negative sign aata hai!

Topic 2 — Integration by Substitution
HOT ▼
Method
Put u = g(x) → du = g'(x)dx
∫f(g(x))·g'(x)dx = ∫f(u)du = F(u) + C = F(g(x)) + C

Key Substitution Forms

Integral FormSubstitutionResult
∫f(ax+b)dxu = ax+b(1/a)·F(ax+b) + C
∫[f(x)]ⁿ·f'(x)dxu = f(x)[f(x)]ⁿ⁺¹/(n+1) + C
∫f'(x)/f(x)dxu = f(x)ln|f(x)| + C
∫sin(ax+b)dxu = ax+b−(1/a)cos(ax+b)+C
∫eᵃˣ dxu = ax(1/a)eᵃˣ + C
∫tan x dxu = cos x−ln|cos x| + C = ln|sec x| + C
∫cot x dxu = sin xln|sin x| + C
⚡ Substitution Identify Karne ka Trick

Agar numerator ≈ denominator ka derivative ho → ln form
Agar [f(x)]ⁿ ke saath f'(x) ho → power form
Agar composite function ho → substitution

Trigonometric Substitutions

When to use which substitution
√(a²−x²) → x = a sinθ
√(a²+x²) → x = a tanθ
√(x²−a²) → x = a secθ

Topic 3 — Integration by Parts
HOT ▼
Integration by Parts Formula
∫u·v dx = u·∫v dx − ∫[(du/dx)·∫v dx] dx

Short form: ∫u dv = u·v − ∫v du
ILATE Rule
Inverse trig · Logarithm · Algebraic · Trig · Exponential
Jo pehle aaye wo u (first function), baaki v (second function)

ILATE Examples

Integralu (ILATE)v
∫x·eˣ dxx (Algebraic)eˣ (Exp)
∫x·sin x dxx (Algebraic)sin x (Trig)
∫ln x dxln x (Log)1 (Algebraic)
∫x²·ln x dxln x (Log)x² (Algebraic)
∫eˣ·sin x dxsin x (Trig)eˣ (Exp)
Special: ∫eˣ[f(x) + f'(x)]dx
∫eˣ[f(x) + f'(x)]dx = eˣ·f(x) + C

Ye formula direct apply karo — Parts nahi lagega!
⚡ eˣ[f(x)+f'(x)] Trick

Agar eˣ ke saath koi function aur uska derivative dono ho → Direct answer = eˣ·f(x)+C. Example: ∫eˣ(sin x + cos x)dx = eˣ sin x + C (kyunki (sin x)' = cos x)

IMPORTANT ▼

Rational function P(x)/Q(x) ko simple fractions mein todke integrate karo.

Types of Partial Fractions

Denominator FormPartial Fraction Form

∫1/(x−a) dx = ln|x−a| + C

A aur B find karne ke liye: Denominator = 0 karo (x ka value nikalo), phir equation mein substitute karo. Example: A/(x−1)+B/(x+2): put x=1 → A milega; put x=−2 → B milega!

Topic 5 — Definite Integrals & Properties
HOT ▼
Definite Integral
∫ₐᵇ f(x)dx = F(b) − F(a) where F'(x) = f(x)

= [F(x)]ₐᵇ = F(b) − F(a) (Newton-Leibniz Rule)

Properties of Definite Integrals (CUET mein Direct Poochhe Jaate Hain!)

Property 1 — Limits interchange
∫ₐᵇ f(x)dx = −∫ᵦₐ f(x)dx

Property 2 — Split limits
∫ₐᵇ f(x)dx = ∫ₐᶜ f(x)dx + ∫ᶜᵇ f(x)dx (a < c < b)

Property 3 — Substitution x → a+b−x
∫ₐᵇ f(x)dx = ∫ₐᵇ f(a+b−x)dx

Property 4 — Substitution x → a−x
∫₀ᵃ f(x)dx = ∫₀ᵃ f(a−x)dx

Property 5 — Even/Odd Functions (Symmetric limits)
f(x) even [f(−x)=f(x)]: ∫₋ₐᵃ f(x)dx = 2∫₀ᵃ f(x)dx
f(x) odd [f(−x)=−f(x)]: ∫₋ₐᵃ f(x)dx = 0

Property 6 — 2a limit
∫₀²ᵃ f(x)dx = 2∫₀ᵃ f(x)dx if f(2a−x) = f(x)
∫₀²ᵃ f(x)dx = 0 if f(2a−x) = −f(x)
⚡ Property 5 Shortcut (Most Used in CUET!)

Symmetric limits (−a to a) → Check f(−x):
Even function → multiply by 2, single side se integrate karo
Odd function → Answer = 0 directly! No calculation needed! 🔥

📌 Property 3 & 4 Use Cases
  • ∫₀^(π/2) sin x / (sin x + cos x) dx → Property 4 (replace x with π/2−x)
  • ∫₀^π x·sin x dx → Property 3 (replace x with π−x)
  • Ye properties direct answer deti hain — calculation bachao!

Topic 6 — Special/Standard Integrals (Must Know)
HOT ▼
Inverse Trig Type Integrals
∫ dx/(x²+a²) = (1/a)tan⁻¹(x/a) + C
∫ dx/√(a²−x²) = sin⁻¹(x/a) + C
∫ dx/√(x²+a²) = ln|x+√(x²+a²)| + C
∫ dx/√(x²−a²) = ln|x+√(x²−a²)| + C
∫ dx/(x²−a²) = (1/2a)ln|(x−a)/(x+a)| + C
∫ dx/(a²−x²) = (1/2a)ln|(a+x)/(a−x)| + C
√ Type Integrals
∫√(a²−x²) dx = (x/2)√(a²−x²) + (a²/2)sin⁻¹(x/a) + C
∫√(x²+a²) dx = (x/2)√(x²+a²) + (a²/2)ln|x+√(x²+a²)| + C
∫√(x²−a²) dx = (x/2)√(x²−a²) − (a²/2)ln|x+√(x²−a²)| + C
Trig Definite Integrals (Walli's Formula)
∫₀^(π/2) sinⁿx dx = ∫₀^(π/2) cosⁿx dx

n even: = [(n−1)(n−3)...1]/[n(n−2)...2] × π/2
n odd: = [(n−1)(n−3)...2]/[n(n−2)...1]

Example: ∫₀^(π/2) sin²x dx = (1/2)×π/2 = π/4
Example: ∫₀^(π/2) sin⁴x dx = (3/4)(1/2)(π/2) = 3π/16

Topic 7 — Area Under a Curve
MOST IMP ▼

Curve y=f(x) aur x-axis ke beech ka area — ye Application of Integrals ka main topic hai.

Area — Basic Formulas
Area between curve y=f(x) and x-axis, from x=a to x=b:
A = ∫ₐᵇ |f(x)| dx = ∫ₐᵇ y dx

Area between curve x=g(y) and y-axis, from y=c to y=d:
A = ∫ᶜᵈ |g(y)| dy = ∫ᶜᵈ x dy

⚠️ Absolute value! If curve below x-axis, take |area|

Area Between Two Curves

Upper curve − Lower curve
A = ∫ₐᵇ [f(x) − g(x)] dx
where f(x) ≥ g(x) in [a,b]

Step 1: Find intersection points (limits a,b)
Step 2: Identify upper curve f(x) and lower curve g(x)
Step 3: Integrate [upper − lower] from a to b
⚡ Step-by-Step Area Method

1. Dono curves ka intersection point nikalo (=limits a, b)
2. Sketch banao (optional but helpful)
3. Upper curve − Lower curve likho
4. Integrate karo limits ke beech
5. Answer positive aana chahiye — agar negative ho to |...| lo

Standard Areas — Direct Formulas (CUET Favourite!)

Areas You Must Know
Circle x²+y²=r²: Area = πr²
Ellipse x²/a²+y²/b²=1: Area = πab
Parabola y²=4ax, from 0 to h: A = (4a/3)(h/4a)^(3/2) × ... → use integration

📌 Quarter circle area = πr²/4

Area of parabola y=x² and line y=x: A = ∫₀¹(x−x²)dx = 1/2−1/3 = 1/6
Area of parabola y²=4x and line x=3: A = 2∫₀³ √(4x)dx = 2·2∫₀³ √x dx = 4·[2/3·x^(3/2)]₀³ = 8√3

Topic 8 — Standard Area Problems (CUET Types)
HOT ▼

Type 1: Parabola and Line

y² = 4ax aur x = h
A = 2∫₀ʰ √(4ax) dx = 2·2√a·∫₀ʰ √x dx
= 4√a · [2x^(3/2)/3]₀ʰ = (8/3)√a · h^(3/2)

y = x² aur y = x (unit parabola)
Intersection: x²=x → x=0,1
A = ∫₀¹(x−x²)dx = [x²/2−x³/3]₀¹ = 1/2−1/3 = 1/6

Type 2: Circle and Line/Parabola

Circle x²+y²=r² — Quarter Area
Area of quarter circle = ∫₀ʳ √(r²−x²) dx
= [x/2·√(r²−x²) + r²/2·sin⁻¹(x/r)]₀ʳ
= 0 + r²/2·sin⁻¹(1) − 0 = r²/2·π/2 = πr²/4 ✓

Ellipse x²/a²+y²/b²=1 — Full Area
A = 4∫₀ᵃ (b/a)√(a²−x²)dx = 4·(b/a)·πa²/4 = πab

Type 3: Area Between Parabola and Circle

y² = x aur x² + y² = 2
Step 1: y²=x and x²+y²=2 → x²+x=2 → x²+x−2=0 → x=1,−2
Since y²=x needs x≥0, intersection at x=1 (y=±1)
Step 2: A = 2[∫₀¹ √x dx + ∫₁^√2 √(2−x²) dx]
(Parabola se x=1 tak, circle se x=1 to √2 tak)
📌 Key Rules for Area Problems
  • Always find intersection points first — ye hi limits hain
  • Symmetric curves ka area → ek side integrate karo, 2 se multiply karo
  • Area negative aa jaye to absolute value lo
  • Parabola y²=4ax → symmetric about x-axis → 2∫₀ʰ √(4ax)dx
  • Area always positive hoti hai — unit sq. mein answer do

⚡ Quick Rules

  • ∫sin = −cos, ∫cos = +sin — negative on sin's side
  • ∫cosec² = −cot, ∫sec² = +tan — negative on cosec's side
  • ILATE rule for integration by parts — L comes before A and T
  • eˣ[f(x)+f'(x)] → Direct = eˣf(x)+C, no parts needed
  • Odd function on (−a,a) → Integral = 0, directly!
  • ∫₀ᵃ f(x)dx = ∫₀ᵃ f(a−x)dx — use this when sin/cos mix ho
  • Area = ∫|f(x)|dx — always positive
  • Two curves area = ∫(Upper − Lower)dx — intersection points = limits
  • Symmetric area = 2 × (half area) — save time!
  • ∫f'(x)/f(x)dx = ln|f(x)|+C — isko pehchano quickly
